コマンド 区分 グラフタイトル 項目名 説明
cpu
mpstat　-P ALL cpu %CPU %user ユーザレベル (アプリケーション) での実行に使用された CPU時間の割合。
mpstat　-P ALL cpu %nice_CPU %nice nice コマンドにて静的優先度を変更したものに関する、ユーザレベルでの実行に使用された CPU時間の割合。
mpstat　-P ALL cpu %system_CPU %sys システムレベル(カーネル)プロセスのCPU使用率。ハードウェア割り込み、ソフト割り込み時間は含まない。
mpstat　-P ALL cpu %iowait_CPU %iowait システムに未完了のディスク I/O 要求がある間の CPU が idle である時間の割合。
mpstat　-P ALL cpu %irq %irq ハードウェア割り込みに使用した時間の割合。
mpstat　-P ALL cpu %softirq %soft ソフト割り込みに使用した時間の割合。
mpstat　-P ALL cpu %idle_CPU %idle CPU が idle で、システムに未完了のディスク I/O 要求がない状態の時間の割合。
mpstat　-P ALL cpu %steal_CPU %steal ハイパーバイザが他の仮想OSに仮想プロセッサを使用し、強制的に待ち時間となっている割合。
mpstat　-P ALL cpu interrupts intr/s 1秒あたりの生成CPU割り込み回数。
sar -u cpu %CPU %user ユーザレベル (アプリケーション) での実行に使用された CPU時間の割合。
sar -u cpu %nice_CPU %nice nice コマンドにて静的優先度を変更したものに関する、ユーザレベルでの実行に使用された CPU時間の割合。
sar -u cpu %system_CPU %system システムレベル(カーネル)プロセスのCPU使用率。ハードウェア割り込み、ソフト割り込み時間は含まない。
sar -u cpu %iowait_CPU %iowait システムに未完了のディスク I/O 要求がある間の CPU が idle である時間の割合。
sar -u cpu %idle_CPU %idle CPU が idle で、システムに未完了のディスク I/O 要求がない状態の時間の割合。
sar -u cpu %steal_CPU %steal ハイパーバイザが他の仮想OSに仮想プロセッサを使用し、強制的に待ち時間となっている割合。
sar -w cpu context_switch/s cswch/s 1秒あたりのコンテキストスイッチ回数。
interrupts
sar -I interrupts interrupts/s intr/s 1秒あたりの生成割り込み数。
memory
sar -r memory free_memory(KB) kbmemfree 利用出来る空きメモリの量 (キロバイト)。
sar -r memory used_memory(KB) kbmemused 使用中のメモリの量 (キロバイト)。カーネル自身が使用しているメモリについては計算に入らない。
sar -r memory %used_memory %memused 使用中メモリの割合。
sar -r memory buffered_memory(KB) kbbuffers カーネルがバッファとして使用中のメモリの量 (キロバイト)。
sar -r memory cached_memory(KB) kbcached カーネルがデータをキャッシュするために使用しているメモリの量 (キロバイト)。
sar -R memory free_memory(num_of_page) frmpg/s 1秒間に、システムが開放したメモリページ数。負の数の場合、システムが割り当てたメモリページ数となる。ページサイズは、マシンアーキテクチャによっては4KBか8KBとなることに注意する。
sar -R memory buffer_memory(num_of_page) bufpg/s 1秒間に、システムがバッファとして追加確保したメモリページ数。負の数の場合、システムによってバッファとして使用されるページ数が減少したことを意味する。
sar -R memory cached_memory(num_of_page) campg/s 1秒間に、システムがキャッシュとして追加確保したメモリページ数。負の数の場合、システムによってキャッシュとして使用されるページ数が減少したことを意味する。
swap
sar -r swap free_swap(KB) kbswpfree 空きスワップ領域の量 (キロバイト)。
sar -r swap used_swap(KB) kbswpused 使用中のスワップ領域の量 (キロバイト)。
sar -r swap %used_swap %swpused 使用中のスワップ領域の割合。
sar -r swap cached_swap_memory kbswpcad キャッシュされたスワップメモリの量 (キロバイト)。これは、一度スワップアウトしてメモリに戻っても、スワップ領域にも残ったままのメモリになる。もしメモリが要求されても、既にスワップ領域にあるため再度スワップアウトの必要はなく、I/O を減らす。
sar -W swap swapin/s pswpin/s 1秒あたりのスワップイン回数。
sar -W swap swapout/s pswpout/s 1秒あたりのスワップアウト回数。
queuelen_loadavg
sar -q queuelen_loadavg run_queue_length runq-sz 実行キューの長さ。(実行可能で実行待ちのキューに入っているプロセス数)
sar -q queuelen_loadavg process_thread_inlist plist-sz プロセスリスト中のプロセス数。
sar -q queuelen_loadavg load_avg_last_min ldavg-1 過去 1 分間のシステムロードアベレージ。この数値は、単位時間当たりに、CPUが割り当てられず待ち状態となっているタスク数を表す。CPUの空きが無い状態に加え、IO待ち状態を含んでいるので、IOバウンドかCPUバウンドかは別の項目で判断すること。一般にはこの数値はCPUコア数（又はその２倍）以下に抑えるべきである。
sar -q queuelen_loadavg load_avg_past_5min ldavg-5 過去 5 分間のシステムロードアベレージ。
sar -q queuelen_loadavg load_avg_past_15min ldavg-15 過去 15 分間のシステムロードアベレージ。
paging
sar -B paging page_in/s pgpgin/s 1 秒あたりのシステムがディスクからページインした合計数 (キロバイト)。カーネル2.2以前ではブロック数。
sar -B paging page_out/s pgpgout/s 1 秒あたりのシステムがディスクにページアウトした合計数 (キロバイト)。カーネル2.2以前ではブロック数。
sar -B paging page_fault/s fault/s 1 秒あたりのシステムが引き起こしたページフォルト (メジャーとマイナー) の回数 (2.5 カーネル以降のみ)。ページフォルトには I/O なしで解決出来るものもあるため、これは I/O を生成するページフォルトの数にはならない。
sar -B paging page_majorfault/s majflt/s 1 秒あたりのシステムが引き起こしたメジャーフォルトの回数で、ディスクからのメモリページのロードを要求したもの (カーネル 2.5 以降のみ)。
sar -B paging page_free/s pgfree/s 1秒あたりにシステムによってフリーリスト上にセットされたページ数。
sar -B paging page_scanned_kswapd/s pgscank/s 1秒あたりにkswapdによってスキャンされたページ数。
sar -B paging page_scanned_direct/s pgscand/s 1秒あたりに直接スキャンされたページ数。
sar -B paging page_steal/s pgsteal/s 1秒あたりに、システムがそのメモリ要求を満たすためキャッシュ（ページキャッシュとスワップキャッシュ）から取り戻したページ数。
sar -B paging %vm_efficiency %vmeff pgsteal / pgscanの値。これはページ変換要求における効率性の測定基準となる。この値が100%に近くなる場合、ほとんど全てのページが、アクティブでないリストの末尾から取られることが繰り返されている時である。この値が低すぎる（30%を切る）場合、仮想メモリが何か問題を抱えている。このフィールドが0の場合、レポートインターバルの間にページがスキャンされていないことを示す。
device
sar -b device diskio_reqcount/s tps 1秒あたりに物理デバイスへ要求されたI/Oリクエスト回数。1回の転送が物理デバイスへの1つのIO要求。複数の論理的なリクエストが一つのリクエストとして結合されてデバイスへ転送される。一つの転送は不確定なサイズとなる。
sar -b device diskio_readcount/s rtps 1秒あたりに物理デバイスへ要求された読み込みリクエスト回数。
sar -b device diskio_writecount/s wtps 1秒あたりに物理デバイスへ要求された書き込みリクエスト回数。
sar -b device diskio_readblocks/s bread/s 1秒あたりにデバイスから読み込んだブロック数。カーネル2.4以降では１ブロック512バイト。それ以前のカーネルでは、ブロックサイズは不確定。
sar -b device diskio_writeblocks/s bwrtn/s 1秒あたりにデバイスへ書き込まれたブロック数。
iostat -x -k -p device merged_read_requests/s rrqm/s １秒毎にマージされた読み込み要求数。
iostat -x -k -p device merged_write_requests/s wrqm/s １秒毎にマージされた書き込み要求数。
iostat -x -k -p device read_requests/s r/s 読み込み要求数/秒。
iostat -x -k -p device write_requests/s w/s 書き込み要求数/秒。
iostat -x -k -p device read_kilobytes/s rkB/s 読み込み量(KB)/秒。
iostat -x -k -p device write_kilobytes/s wkB/s 書き込み量(KB)/秒。
iostat -x -k -p device avg_request_size avgrq-sz I/O要求の平均サイズ(セクタ)。
iostat -x -k -p device avg_request_queuelen avgqu-sz I/O要求の平均待ち数。
iostat -x -k -p device IO_avg_time await I/O要求の平均待ち時間 (msec) 。
iostat -x -k -p device IO_avg_servicetime svctm I/O要求の平均処理時間 (msec) 。
iostat -x -k -p device %CPU_util_IO %util I/O要求におけるCPU消費量(%) 。
sar -d -p device transfer/s tps 1秒あたりに指定デバイスへ要求されたI/Oリクエスト回数。1回の転送が物理デバイスへの1つのIO要求。複数の論理的なリクエストが一つのリクエストとして結合されてデバイスへ転送される。一つの転送は不確定なサイズとなる。
sar -d -p device rw_sectors/s rd_sec/s デバイスから読み込まれたセクタ数。セクタサイズは512バイト。
sar -d -p device rw_sectors/s wr_sec/s デバイスに書き込まれたセクタ数。セクタサイズは512バイト。
sar -d -p device avg_request_size avgrq-sz デバイスへの平均リクエストサイズ（セクタ）。
sar -d -p device avg_request_queuelen avgqu-sz デバイスへの平均リクエストキューサイズ。
sar -d -p device IO_avg_time await デバイスへのI/O要求が処理されるまでに掛かった時間（msec）。これは、キュー内でリクエストによって使われた時間と、その処理時間を示す。
sar -d -p device IO_avg_servicetime svctm デバイスへのI/O要求が処理されるまでの平均時間（msec）。
sar -d -p device %CPU_util_IO %util デバイスへのI/O要求中の平均CPU使用率。
vmstat
vmstat vmstat runtime_wait_process r 実行待ちのプロセス数。
vmstat vmstat uninterruptible_sleep_process b 割り込み不可能なスリープ状態にあるプロセス数。
vmstat vmstat vertual_memory swpd 仮想メモリの総量 (KB)。
vmstat vmstat free_memory free 空きメモリの量 (KB)。
vmstat vmstat buffer_memory buff バッファに用いられているメモリの量 (KB)。
vmstat vmstat cache_memory cache キャッシュに用いられているメモリの量
vmstat vmstat memory_swap_in/s si ディスクからスワップインされているメモリの量 (KB/s)
vmstat vmstat memory_swap_out/s so ディスクにスワップしているメモリの量 (KB/s)。
vmstat vmstat recieved_from_device(blocks/s) bi ブロックデバイスに送られたブロック (blocks/s)。
vmstat vmstat send_to_device(blocks/s) bo ブロックデバイスから受け取ったブロック (blocks/s)。
vmstat vmstat interrupts/s in 一秒あたりの割り込み回数。クロック割り込みも含む。
vmstat vmstat context_switches/s cs 一秒あたりのコンテキストスイッチの回数。
vmstat vmstat user_and_nice_time us ユーザレベル (アプリケーション) での実行に使用された CPU時間の割合。niceにより静的優先度を変更されたプロセスのCPU時間を含む。
vmstat vmstat system_time sy システムレベル(カーネル)プロセスのCPU使用率。
vmstat vmstat idle_time id CPUが idle 状態である時間。カーネル 2.5.41 以前では、IO 待ち時間を含んでいる。
vmstat vmstat iowait_time wa CPUが IO 待ちに消費した時間。Linux 2.5.41 以前では、idle時間を含んでいる。
vmstat vmstat stolen_by_vm_time st 仮想OSに仮想プロセッサを使用し、強制的に待ち時間となっている割合。Linux 2.6.11より前では未知。
network
sar -n DEV network recieve_packets/s rxpck/s 1 秒あたりにネットワークデバイスが受信したパケット数の合計。
sar -n DEV network transfer_packets/s txpck/s 1 秒あたりにネットワークデバイスが送信したパケット数の合計。
sar -n DEV network TR_TX_bytes(KB/s) rxkB/s 1 秒あたりにネットワークデバイスが受信したバイト数の合計（キロバイト）。
sar -n DEV network TR_TX_bytes(KB/s) txkB/s 1 秒あたりにネットワークデバイスが送信したバイト数の合計（キロバイト）。
sar -n DEV network recieve_cmp_packets/s rxcmp/s 1 秒あたりにネットワークデバイスが受信した圧縮パケット数の合計。
sar -n DEV network transfer_cmp_packets/s txcmp/s 1 秒あたりにネットワークデバイスが送信した圧縮パケット数の合計。
sar -n DEV network recieve_cmp_multi_packets/s rxmcst/s 1 秒あたりにネットワークデバイスが受信したマルチキャストパケット数の合計。
sar -n EDEV network recieve_bad_packets/s rxerr/s 1 秒あたりにネットワークデバイスが受信した不良パケット数の合計。
sar -n EDEV network transfer_bad_packets/s txerr/s 1 秒あたりにネットワークデバイスにおいてパケット送信中に発生したエラー数の合計。
sar -n EDEV network collision_transpackets/s coll/s 1 秒あたりのパケット送信中に発生したコリジョン数。
sar -n EDEV network drop_recpackets_overbuffer/s rxdrop/s 1 秒あたりの、Linuxカーネルバッファ内の領域不足によってネットワークデバイスが破棄した受信パケット数。
sar -n EDEV network drop_transpackets_overbuffer/s txdrop/s 1 秒あたりの、Linuxカーネルバッファ内の領域不足によってネットワークデバイスが破棄した送信パケット数。
sar -n EDEV network carrier_error_transpackets/s txcarr/s 1 秒あたりの、ネットワークデバイスにおいてパケット送信中に発生したキャリアエラー数。
sar -n EDEV network frame_align_error_recpackets/s rxfram/s 1 秒あたりの、ネットワークデバイスの受信パケットについて発生したフレームアライメントエラー数。なお、フレームアライメントエラーとは、フレーム内のビット数が 8 で割り切れない時に出されるエラーのこと。何らかの理由によりビット落ちが発生した場合に生じる。
sar -n EDEV network FIFO_overrun_erro_recpackets/s rxfifo/s 1 秒あたりの、ネットワークデバイスの受信パケットについて発生した FIFO オーバーランエラー数。バッファ不足でパケットをバッファに取り込めない時にセットされる。
sar -n EDEV network FIFO_overrun_erro_transpackets/s txfifo/s 1 秒あたりの、ネットワークデバイスの送信パケットについて発生した FIFO オーバーランエラー数。バッファ不足でパケットをバッファに取り込めない時にセットされる。
socket
sar -n SOCK socket total_sockets totsck 使用されている合計ソケット数。
sar -n SOCK socket tcp_sockets tcpsck 使用されているTCPソケット数。
sar -n SOCK socket udp_sockets udpsck 使用されているUDPソケット数。
sar -n SOCK socket raw_sockets rawsck 使用されているRAWソケット数。RAWソケットについては、man 2 socket参照。
sar -n SOCK socket ip-frag ip-frag 現在のIPフラグメント数。フラグメントは、MTUを超えるIPパケット送信時に発生するIPパケットの分割である。
sar -n SOCK socket tcp-timewait tcp-tw 現在のTIMEWAIT状態であるTCPソケット数。
nfs/client
sar -n NFS nfs/client RPC_calls call/s 1秒あたりのRPC生成回数。
sar -n NFS nfs/client RPC_retransmitted retrans/s 1秒間に必要になったRPC再送要求回数。（タイムアウトなどの理由によるもの）
sar -n NFS nfs/client RPC_read read/s 1秒あたりのRPC[read]がコールされた回数。
sar -n NFS nfs/client RPC_write write/s 1秒あたりのRPC[write]がコールされた回数。
sar -n NFS nfs/client RPC_access access/s 1秒あたりのRPC[access]がコールされた回数。
sar -n NFS nfs/client RPC_getattr getatt/s 1秒あたりのRPC[getattr]がコールされた回数。
nfs/server
sar -n NFSD nfs/server RPC_recieve_calls scall/s 1秒あたりのRPC受信回数。
sar -n NFSD nfs/server RPC_error_rec_calls badcall/s 1秒間に、受信したRPCがエラーを起こした回数。
sar -n NFSD nfs/server recieve_packets packet/s 1秒間に受信したネットワークパケット数。
sar -n NFSD nfs/server recieve_udp_packets udp/s 1秒間に受信したUDPパケット数。
sar -n NFSD nfs/server recieve_tcp_packets tcp/s 1秒間に受信したTCPパケット数。
sar -n NFSD nfs/server reply_cache_hits hit/s 1秒間に、応答キャッシュにヒットした回数。
sar -n NFSD nfs/server reply_cache_miss miss/s 1秒間に、応答キャッシュがミスした回数。
sar -n NFSD nfs/server RPC_recieve_read sread/s 1秒間に受信したRPC[read]コールの回数。
sar -n NFSD nfs/server RPC_recieve_write swrite/s 1秒間に受信したRPC[wite]コールの回数。
sar -n NFSD nfs/server RPC_recieve_access saccess/s 1秒間に受信したRPC[access]コールの回数。
sar -n NFSD nfs/server RPC_recieve_getattr sgetatt/s 1秒間に受信したRPC[getattr]コールの回数。
ktables
sar -v ktables directory_cache_unused dentunusd 使用されていないディレクトリキャッシュエントリ。
sar -v ktables file_handlers file-nr ファイルハンドラ数。
sar -v ktables inode_handlers inode-nr inodeハンドラ数。
sar -v ktables pseudo_terminals pty-nr 仮想ターミナル数。
process
sar -c process process_create/s proc/s 1秒あたりのプロセス生成数。
pidstat -u process user_cpu_pct %user プロセスが使用したユーザレベルCPU使用時間のパーセント表示。（niceにより静的優先度を変更されたものを含む）
pidstat -u process system_cpu_pct %system プロセスが使用したシステム(カーネル)レベルCPU使用時間の割合。
pidstat -u process total_cpu_pct %CPU タスクによって使用された総合CPU時間の割合。SMP環境で-Iオプションを使用した場合、CPUの合計数によって割った数値になる。
pidstat -u process attached_processor CPU 割り当てられているプロセッサ（CPUコア）番号。
pidstat -r process minor_fault/sec minflt/s 1秒間にプロセスが生成したマイナーフォルト数。ディスクからメモリページを要求することはない。
pidstat -r process major_fault/sec majflt/s 1秒間にプロセスが生成したメジャーフォルト数。ディスクからメモリページを要求する。
pidstat -r process virtual_memory(KB) VSZ 仮想メモリサイズ。タスク全体で使用している仮想メモリサイズ（キロバイト）。
pidstat -r process resident_set_memory(KB) RSS 常駐セットサイズ。タスクによって使用されているスワップでない物理メモリ領域（キロバイト）。
pidstat -r process memory_usage_pct %MEM プロセスが使用する全体に対するメモリ使用率と考えられるが、詳細は不明。（pidstatのマニュアルに未記載の項目）
pidstat -d process read_disk(KB) kB_rd/s 1秒間にタスクが引き起こしたディスクからのreadサイズ(キロバイト)。
pidstat -d process write_disk(KB) kB_wr/s 1秒間にタスクが引き起こしたディスクへのwriteサイズ(キロバイト)。
pidstat -d process cancel_write(KB) kB_ccwr/s 1秒間にタスクによってキャンセルされたディスクへの書き込み（キロバイト）。タスクが幾つかの汚れたページキャッシュを切り捨てる時に発生する。このケースにおいて、別のタスクが引き起こしたIOで、実際には生じないものが発生する。
tty

sar -y tty transmit_interrupts xmtin/s 現在のシリアルラインに送信した割り込み数。
sar -y tty frame_errors framerr/s 現在のシリアルラインで発生したフレームエラー数。
sar -y tty parity_errors prtyerr/s 現在のシリアルラインで発生したパリティエラー数。
sar -y tty break brk/s 現在のシリアルラインで発生したbreak数。
sar -y tty overruns ovrun/s 現在のシリアルラインで発生したオーバランエラー数。
